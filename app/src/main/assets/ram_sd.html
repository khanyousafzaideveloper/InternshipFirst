Fix SD Card by Formatting a RAW SD card to FAT32

An SD card may seem to be corrupted if it has no file system on it. Such an SD card is typically referred to by experts as RAW, and most operating systems won’t display it as an available storage device and allow you to write files to it.

Since a RAW SD card shouldn’t have any data on it, formatting it to FAT32 is a relatively safe operation. Just make sure to select the RAW SD and not some other storage device.

To format a RAW SD card to FAT32:

  1. Connect the SD card to your computer.
  2. Type “cmd” in the search box on the taskbar and select Run as administrator.
  3.  pixxxxxxxx
  4. Type diskpart and press Enter on your keyboard
  5. Type list volume and press Enter.
  6. Type select volume <the number corresponding to the RAW SD card> and press Enter.
  7. Type format fs=fat32 and hit Enter.

Diskpart is a useful command that can do a lot more than format RAW partitions, as we explain in the next chapter.
