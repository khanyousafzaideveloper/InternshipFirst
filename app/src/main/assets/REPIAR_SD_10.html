Format to Repair Corrupted SD Card

Formatting is a drastic way how to fix a corrupted SD card, but it can quickly and effectively replace a faulty partition table with a new one and recreate a damaged file system.

While there are many different ways how to format an SD card, they all accomplish the same goal, so there’s no reason to use unnecessarily complicated third-party tools since the formatting tool in Windows is more than enough to get the job done.

To format a corrupted SD card:

  1. Type “this PC” in the search on the taskbar and select This PC.

  3. Right-click on your SD card and select the Format… option.(pixxxxxxx)
  4. Select the desired File system. You can leave the allocation unit size as it is.
  5. Make sure the Quick Format option is not checked.
  6. Click Start to format the SD card.(PIXXXXXXXXXX)

The difference between Quick Format and Full Format is that the latter scans the formatted device for bad sectors, while the former only removes address files from it.
