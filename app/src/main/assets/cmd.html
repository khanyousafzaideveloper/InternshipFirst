Use the Diskpart Command in CMD (without Formatting)

Not being afraid of the command line pays off because it means that you can use powerful tools like Diskpart to fix SD card issues without formatting.

When using Diskpart, keep in mind that you must first select the correct drive and then apply commands to it. As a safety measure, we recommend you disconnect as many drives as possible to minimize the possibility of selecting the wrong one.

To fix an SD card using Diskpart:

  1. Connect the SD card to your computer.
  2. Type “cmd” in the search box on the taskbar and select Run as administrator.

  4. Type diskpart and press Enter on your keyboard.
  5. Type list disk and press Enter.
  6. Type select disk <the number corresponding to your SD card> and press Enter.
  7. Type recover and hit Enter. This command will refresh the state of the SD card.
